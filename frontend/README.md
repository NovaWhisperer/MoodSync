# Frontend (MoodSync)

This folder contains the React + Vite client for MoodSync.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build production assets
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Environment

Create `.env` from `.env.example`:

```bash
VITE_API_BASE_URL=https://your-render-service.onrender.com
```

`VITE_API_BASE_URL` is required. The app does not use a hardcoded localhost fallback.

For full setup instructions, see the root README.
